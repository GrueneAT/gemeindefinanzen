      Wichtige Einordnung

      Diese Auswertung ist eine Suchhilfe, keine Empfehlung. Sie zeigt, wo im Budget Betraege welcher Groessenordnung ueberhaupt liegen. Ob ein Posten kuerzbar ist, ist eine fachliche und politische Entscheidung — jede Zeile braucht eine eigene Bewertung.
